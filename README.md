# GioRiRao
Đồ án tốt nghiêp, quản lý nước, và bán nước

# Hướng dẫn sử dụng dự án

## Cài đặt các package cần thiết
npm i

## Thêm các biến môi trường cần thiết để chạy các ứng dụng thứ ba và phần mềm
CLOUDINARY_API_SECRET
NODE_ENV
DEV_APP_PORT
DEV_DB_PORT
DEV_DB_HOST
DEV_DB_NAME
REDIS_PORT
REDIS_HOST
REDIS_PASSWORD

## Chạy phần mềm
npm run dev
